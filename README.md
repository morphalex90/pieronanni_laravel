# Piero Nanni v3 - Laravel

## How to install
`composer install` then `npm install`

## How to start
`npm run dev` then use Herd or `php artisan serve`

### Google fonts
    $ php artisan google-fonts:fetch

## Pint
    $ ./vendor/bin/pint

## Generate it all
    $ php artisan make:model Post -mcrf
