# Piero Nanni v3 - Laravel

## How to install & run
    composer install
    npm install
    php artisan storage:link
    composer run dev

### Google fonts
    $ php artisan google-fonts:fetch

## Pint
    $ ./vendor/bin/pint

## Generate it all
    $ php artisan make:model Post -mcrf
